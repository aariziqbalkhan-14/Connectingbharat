Connect instantly. Chat privately. Leave no trace.
Connecting Bharat is a privacy-first, account-free communication platform that lets two people connect instantly using a one-time secret code. Messages are end-to-end encrypted and exchanged directly between browsers. No chat history, user accounts, or server-side message storage—just secure, temporary peer-to-peer communication.
# ConnectingBharat

## Features
- No login required
- One-time secret room code
- End-to-end encrypted chat
- Peer-to-peer calling (PeerJS/WebRTC)
- No chat history stored
- Browser-to-browser communication

## Tech Stack
- HTML
- CSS
- JavaScript
- PeerJS
- WebRTC

## Future Plans
- File sharing
- Voice messages
